# Сравниваем вакансии программистов

Скрипт, при помощи которого можно сравнить количество вакансий для программистов и средние заработные платы на сайтах HeadHunter и SuperJob.

### Как установить

**Python3.13** должен быть уже установлен. 

Скачайте код с **GitHub**. Затем установите зависимости.

```sh
pip install -r requirements.txt
```

### Переменные окружения

Часть настроек проекта берётся из переменных окружения. Чтобы их определить, создайте файл `.env` рядом с `main.py` и запишите туда данные в таком формате: `ПЕРЕМЕННАЯ=значение`.

Доступна 1 переменная:
`SUPERJOB_SECRET_KEY` — ключ приложения, который можно получить при регистрации на SuperJob >>>
[регистрация](https://api.superjob.ru/register/)

Переменная передается в `headers` при каждом запросе к *SuperJob API*:
```python
headers = {
        "Host": "api.superjob.ru",
        "X-Api-App-Id": secret_key,
        "Content-Type": "application/x-www-form-urlencoded",
    }
```

### Пример запуска программы:
```python
python3.13 main.py
```

### Примечание:

По умолчанию программа вычисляет среднюю заработную плату по 10 самым популярным вакансиям для программистов.

Для изменения перечня сравниваемых вакансий достаточно скорректировать список `PROFESSIONS` в программном коде.
